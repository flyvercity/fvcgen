1. Генератор сценариев (далее Генератор) должен создавать fvc файл для объектов согласно файла конфигурации.
2. Файл конфигурации должен содержать общий раздел и раздел настроек объектов
3. Общий раздел должен содержать:
	- географические координаты базовой точки сценария (для пересчета координат из ENU)
	- значения по умолчанию (применяются для всех объектов для которых не указаны явно):
		- скорость объекта
		- высота объекта
		- МО и СКО ошибки определения координат (по трем осям траекторной СК, (ориентированной по касательной к траектории) затем пересчитать их в ENU)
		- шаг по времени обновления координат (сек)
	- путь и имя выходного файла (опционально)
4. Раздел настроек объектов должен содержать:
	- список origins, каждый включает:
		- значения по умолчанию аналогичные указаным выше, но применимые в рамках этого origin
		- список объектов, каждый включает:
			- идентификатор(ы) объекта
			- значения по умолчанию аналогичные указаным выше, применимые для объекта (опционально)
			- задержка старта объекта в сценарии (сек) (опционально, если нет то 0)
			- список ППМ объекта в СК ENU, содержащий:
				- координаты ENU относительно базовой точки, метры
				- скорость объекта на участке от этого до следующего ППМ (опционально)
5. Координаты объекта должны расчитываться в соотвествии с кинематическими уравнениями движения материальной точки.
6. (Опционально!) Изменение траектории движения и скорости при переходе через ППМ сделать плавным
7. Координаты объекта должны определяться как нормально распределенная несмещенная случайная величина с заданными параметрами распределения (мат. ожидание и СКО - заданы в настройках)
8. (Опционально!) Добавить возможность полета по закольцованному маршруту, где последний ППМ из списка соединятеся с первым (нужно будет добавить настройку)
9. Предусмотреть возможность задания имени и пути выходного файла в параметрах командной строки и в файле конфигурации
10. Предусмотреть возможность генерации шаблона файла настроек в параметрах командной строки
11. (Опционально!) Сделать потоковую выдачу данных для передачи во fusion
